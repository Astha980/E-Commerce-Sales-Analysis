# E-Commerce Sales Analysis
## Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset using Python and Excel.
The objective was to identify sales trends, analyze customer reviews, compare product categories and regions, and build a business dashboard.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Excel
- Google Colab
## Project Workflow
1. Data Cleaning
2. Data Merging
3. Sales Analysis
4. Data Visualization
5. Dashboard Creation
6. Business Insights Report
## Key KPIs
- Total Revenue
- Total Orders
- Top Product Category
- Best Sales Month
## Visualizations
- Revenue by Category
- Monthly Sales Trend
- Regional Sales
- Order Value Distribution
- Customer Review Distribution
- Heatmap
